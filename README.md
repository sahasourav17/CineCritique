# CineCritique

A web application written in React and Node.js that allows users to get movie details, review them, and search for them by genre, name, and so on.

## Technologies used

### Backend:

- Nodejs
- Express
- DynamoDB
- MongoDB

### Frontend:

- React
- Tailwind
